# Hướng dẫn sử dụng

## Backend (.NET)

### Yêu cầu
- .NET Core SDK
- SQL Server

### Cài đặt và chạy
1. **Chạy Migration:**
	update-database
2. **Chạy Backend:**
	```bash
   run
   ```
## Frontend (ReactJS)

### Yêu cầu
- Node.js

### Cài đặt và chạy

1. **Cài đặt dependencies:**
	```bash
   npm install
   ```
2. **Chạy ứng dụng:**
	```bash
   npm run dev
   ```
   
## Lưu ý
- Cập nhập localhost với địa chỉ trong máy
